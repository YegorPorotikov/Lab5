 Як зробити внесок у Lab5

Ми раді, що ви хочете допомогти! Будь-який внесок вітається.

 Звіти про помилки (Bug Reports)

Якщо ви знайшли помилку, будь ласка, переконайтеся, що вона ще не описана в розділі [Issues](https://github.com/YourUsername/YourProjectName/issues).

Якщо це нова помилка, будь ласка, створіть нове Issue, вказавши:
 Що ви робили.
 Що ви очікували побачити.
 Що сталося насправді.

 Внесок кодом (Pull Requests)

Найкращий спосіб зробити внесок — це через Pull Requests.

1. Зробіть Fork репозиторію.
2. Створіть нову гілку для ваших змін (`git checkout -b feature/my-new-feature`).
3. Зробіть ваші зміни** та не забудьте зробити "атомарні" комміти (`git commit -m 'Feat: Add some feature'`).
4. Відправте зміни у ваш форк (`git push origin feature/my-new-feature`).
5. Відкрийте Pull Request в оригінальний репозиторій.

Дякуємо за ваш внесок!
